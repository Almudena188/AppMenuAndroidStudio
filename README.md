# 📋 ENUNCIADO
Haz un programa que tenga un menú y que sea capaz de: 
* Abrir un enlace de internet
* Abrir la camara y hacer una foto
* Enviar un emai
* Llamar a un nº de teléfono
* Buscar en los contactos
* Abrir una dirección en el google maps por coordenadas
* Tener un apartado de información del creador
